## 📖 About The Project

**Personal Profile App** is a desktop application built using C# and Windows Forms that displays personal information in a clean and organized UI.

The project includes:
- Home profile screen
- About Me section
- Contact form with validation
- External links to social profiles

---

## ✨ Features

- Modern and clean UI design
- Multiple forms navigation (Show / ShowDialog)
- Input validation using MessageBox
- Interactive Contact form
- External links (GitHub / LinkedIn)
- Keyboard navigation using TabIndex & TabStop
- Profile image display using PictureBox

## 🚀 How It Works

1. Launch the application
2. The main profile screen will appear
3. Click **About Me** to view personal details
4. Click **Contact** to send a message
5. Use **Exit** to close the application
